# 1st-Place-MIS-Project
1st Place, The University of Texas at Austin McCombs School of Business MIS C# Development Competition

## Overview
Developed a full-stack web application using C# and ASP.NET to support business operations, including inventory management, reporting, and customer workflows. Built as part of a competitive, team-based MIS development project.

## Key Features
- Admin reporting dashboard with profit margin analysis  
- Inventory and procurement management system  
- Customer ordering and cart functionality  
- Email confirmation and notification system  

## Tech Stack
- C#  
- ASP.NET MVC  
- Entity Framework Core  
- SQL Server  
- HTML, CSS, JavaScript  

## My Role
- Developed backend logic and application functionality using C#  
- Built reporting and data processing features for business insights  
- Collaborated within a team using Git and GitHub  

## Running the Project
This project uses a SQL Server database for full functionality.

Due to database access restrictions (credentials and environment-specific configuration), the application may not run locally without additional setup.

To run this project locally, you would need to:
- Configure a SQL Server instance  
- Update the connection string in `appsettings.json`  
- Recreate the database using Entity Framework migrations  

Despite this, all application logic, structure, and features are fully implemented and can be reviewed directly in the codebase.
